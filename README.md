# 🧪 CMSE326 – Web Testing & Quality Assurance Project

## 📘 Course
**CMSE326 – Software Quality Assurance and Testing**  
**Eastern Mediterranean University (EMU)**

## 📝 Project Description
This project focuses on the quality assessment and web testing of a real-world news website as part of the CMSE326 course. The testing process includes functional testing, UI/UX inspection, usability evaluation, bug tracking, and performance checks.

Our aim is to ensure the target website meets expected standards of reliability, responsiveness, and user experience.

## 🌐 Website Under Test
- [Kıbrıs Gazetesi](https://www.kibrisgazetesi.com)

## 🔧 Testing Methods Used
- Manual Functional Testing
- Usability Testing
- UI Consistency Checks
- Accessibility Testing
- Google Lighthouse Audits
- Bug Reporting and Documentation
